KEY DESIGN PRINCIPLES EXTRACTED
================================================================================

1. COLOR-CODED ACTIONS
   - Teal = Include filters, primary actions
   - Orange = Exclude filters, warnings, delete
   - Yellow = Highlights, suggestions
   - Gray = Text, borders, disabled states

2. HOVER STATES
   - Most elements have smooth color transitions
   - Use onMouseEnter/onMouseLeave for interactive feedback
   - Apply transition-colors or transition-all

3. GRADIENT ACCENTS
   - Used for logos, branding, emphasis
   - Blue to purple gradient for brand elements

4. WHITE CARDS ON GRAY BACKGROUND
   - Creates clear visual hierarchy
   - Cards have subtle shadows that grow on hover

5. CONSISTENT SPACING
   - 4px baseline unit throughout
   - Proportional scaling (1x, 2x, 3x, 4x, 6x)

6. ICON + TEXT PATTERNS
   - Icons used with text for clarity
   - Icons colored to match content type

7. INLINE STATUS BADGES
   - Color-coded type indicators
   - Used extensively for filter types

8. DARK MODE SUPPORT
   - CSS variables in place
   - Light/dark mode ready

9. RESPONSIVE DESIGN
   - Mobile-first approach
   - Breakpoints: mobile, tablet (sm), desktop (md), large (lg)

10. ACCESSIBILITY FIRST
    - Semantic HTML throughout
    - Focus states clearly visible
    - Proper contrast ratios
